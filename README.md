# Hyperspectral Gas Detection Using Spectral-Spatial Feature Learning

This project focuses on detecting and classifying gas emissions from hyperspectral images using advanced deep learning techniques such as **3D Convolutional Neural Networks (3D-CNN)** and **Autoencoders**. The system extracts spectral-spatial features from hyperspectral data to improve gas identification accuracy and environmental monitoring.

---

## Features

* Detection of gases from hyperspectral image datasets.
* Spectral-spatial feature extraction using deep learning.
* 3D-CNN architecture for accurate gas classification.
* Autoencoder-based dimensionality reduction and feature learning.
* Data preprocessing and visualization support.
* Improved detection accuracy compared to traditional methods.

## Usage

1. Clone the repository from GitHub.
2. Install all required dependencies.
3. Load the hyperspectral dataset.
4. Train the model using the provided scripts.
5. Test the model for gas detection and classification.
6. Visualize prediction results and performance metrics.
